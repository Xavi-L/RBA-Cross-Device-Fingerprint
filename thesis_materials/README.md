# 毕业论文与投稿材料目录

本目录保存 HybridGuard 毕业论文成品、章节草稿、参考文献材料和后续投稿可复用的论文图表。学校流程材料、旧版修复备份和提交前备份已移动到 `archive/graduation_submission/thesis_materials/`。

## 成品与模板

- `梁骁-毕业论文.docx`：毕业论文最终 Word 成品，也是后续整理投稿论文时最完整的中文来源。
- `梁骁-毕业论文.pdf`：毕业论文最终 PDF 导出版本，用于版式和页码核对。
- `北航本科生论文模板.docx`：学校本科论文模板，保留作格式参考。

## 章节草稿

- `chapter1_introduction.md`：第 1 章“绪论”正文草稿。
- `chapter2_system_design.md`：第 2 章“系统设计与总体架构”正文草稿。
- `chapter3_module_implementation.md`：第 3 章“关键模块设计与实现”正文草稿。
- `chapter4_experiment_analysis.md`：第 4 章“实验设计与结果分析”正文草稿，依据 `ablation/` 分组实验和图表材料撰写。
- `chapter4_experiment_analysis_compressed.md`：第 4 章压缩版草稿。
- `chapter5_conclusion.md`：第 5 章“结论与展望”正文草稿。

## 参考文献与审阅材料

- `candidate_references.md`：候选参考文献库，适合继续筛选为投稿论文参考文献。
- `actual_references_gbt.md`：毕业论文正文实际引用文献的 GB/T 格式列表，按正文首次出现顺序排列。
- `chinese_reference_candidates.md`：中文参考文献补充候选。
- `口语化表述检查建议.md`：毕业论文文字审阅记录，保留作中文表述复核参考。

## 图表材料

- `top_journal_figures/`：可继续复用或重绘的系统架构、离线标注训练链路、端侧评分链路、跨层一致性分析和端侧模块结构图，包含 PNG 与 SVG 版本。

后续若要改写为投稿论文，建议以 `梁骁-毕业论文.docx`、`chapter4_experiment_analysis.md`、`ablation/README.md` 和 `top_journal_figures/` 为主线材料，学校表格和答辩备份只从 `archive/graduation_submission/` 查找。
