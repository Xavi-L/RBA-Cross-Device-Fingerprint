# 毕业论文材料目录

本目录用于保存毕业论文写作过程中的材料，包括候选参考文献、文献综述笔记、章节草稿、图表说明和引用格式整理。

当前文件：

- `candidate_references.md`：第一轮联网检索得到的候选参考文献库，后续需要继续精读、筛选和转成正式引用格式。
- `actual_references_gbt.md`：第1章至第5章正文中实际引用文献的 GB/T 格式列表，按正文首次出现顺序排列。
- `chapter1_introduction.md`：第1章“绪论”正文草稿，引用位置使用醒目的候选文献编号占位。
- `chapter2_system_design.md`：第2章“系统设计与总体架构”正文草稿，包含 Mermaid 架构与流程示意图。
- `chapter3_module_implementation.md`：第3章“关键模块设计与实现”正文草稿，依据当前 Android、后端、标注、训练和端侧评分代码撰写。
- `chapter4_experiment_analysis.md`：第4章“实验设计与结果分析”正文草稿，依据根目录第四章大纲和 `ablation/` 实验结果撰写。
- `chapter5_conclusion.md`：第5章“结论与展望”正文草稿，对全文工作、实验结论和后续方向进行简要总结。
